# DingLab气象研究中心综合信息服务平台

基于 React + Vite + TypeScript 重建的气象信息集成系统。

## 技术栈

- React 18
- Vite 5
- TypeScript
- react-router-dom

## 功能特性

- 顶部 Header 显示标题和实时时间
- 左侧导航菜单（支持折叠/展开）
- 右侧 iframe 内容区域
- 本地页面和外部网站混合导航
- 外部链接在新标签页打开
- 管理员后台（导航管理）
- 密码修改功能

## 安装步骤

```bash
npm install
npm run dev
```

## 构建命令

```bash
npm run build
```

## 预览命令

```bash
npm run preview
```
